RYGNECO REPORTING DEMONSTRATION
===============================

STATUS
------
This is a product-program prototype with fictional sample data.
It is not an operating report or customer deliverable.

WINDOWS
-------
1. Double-click START_PROJECT.bat.
2. Open http://localhost:3000 in a browser.
3. Review the labeled demonstration interface.

COMMAND LINE
------------
1. Open a terminal in this repository.
2. Run: node run-demo.js
3. Open: http://localhost:3000

STOP
----
Press Ctrl+C in the terminal.

The standalone page does not require npm install. The module source remains a
prototype archive and requires implementation, security, data-governance, and
acceptance work before any production integration.
