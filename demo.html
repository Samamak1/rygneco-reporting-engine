💻 Easy Integration

Integrate the reporting module into your existing dashboard with just a few lines of code:

// Import and initialize the reporting module
import { ReportingModule } from 'ewaste-reporting-module';

const reporting = new ReportingModule({
  apiEndpoint: 'https://your-api.com',
  clientId: 'your-client-id',
  theme: 'default'
});

// Generate a quarterly report
const report = await reporting.generateReport('quarterly', {
  sections: {
    executiveSummary: true,
    environmentalImpact: true,
    financialImpact: true
  }
});
